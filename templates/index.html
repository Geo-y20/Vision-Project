How to Use

  • For object detection, navigate to the Object Detection page, press "Capture and Detect" to capture an image and hear detected objects.
  • For OCR, navigate to the OCR page, upload an image, and get the text read out loud.
  • For face recognition, choose to collect a new dataset or start face recognition.
